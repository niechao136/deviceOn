# 文本确认判断提示词

**任务**  
判断输入文本是否为确认/肯定性表述，返回JSON格式结果。

**输入文本**  
{{#context#}}

**判断标准**
1. 包含明确同意/确认词汇（如：好、好的、确定、可以、没问题、是的、OK、Go）
2. 表达正向完成态（如：已处理、已完成）
3. 积极回应请求（如：马上办、这就做）
4. 排除否定/模糊表述（如：不行、可能吧、再说）

**处理流程**
1. 文本解析：提取核心动词/表态词
2. 语义匹配：对照确认表达词库
3. 语境分析：识别应答对象完整性
4. 双重验证：交叉检查否定词存在性

**输出要求**
```json
{
  "judgment": "<boolean>",
  "reason": "<不超过15字的关键词依据>"
}
```
