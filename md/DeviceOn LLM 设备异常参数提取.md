# 信息提取提示词

## 输入数据
- `问题`: {{#context#}}
- `设备数组`:
    ```json
    {{#1744082994767.device#}}
    ```

---

## `设备数组`结构说明：
```markdown
- **顶层：** 设备数组，每个设备包含：
    - **"id"：** 设备ID
    - **"nm"：** 设备名称
    - **"hw_n"：** 硬件异常数量
    - **"sw_n"：** 软件异常数量
    - **"pp_n"：** 周边外设异常数量
    - **"ud_n"：** 上下线异常数量
    - **"bt_n"：** 电池异常数量
    - **"sc_n"：** 设备安全异常数量
    - **"er_n"：** 异常总数量
```

---

## 设备筛选
- **请根据`设备列表`中每个设备的异常数量和`问题`文本的描述来筛选设备**
- **输出的`device`请根据`问题`涉及的异常数量进行排序**

---

## 输出格式
输出 JSON 格式必须符合以下结构：
```json
{
  "device": [
    {
      "id": "<设备ID，请保证id一定存在并且对应的设备正确>",
      "name": "<设备名称，请保证id与name对应关系正确>",
      "reason": "<涉及异常的数目以及顺序>"
    }
  ]
}
```
